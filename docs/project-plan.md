# Project Plan

The project runs over eight weeks and is split into five phases:

1. **Phase 1 – Data Centre Selection (Week 1)**  
   Decide the cloud deployment type, select the region, and choose a suitable network topology.

2. **Phase 2 – Requirement Gathering (Week 2)**  
   Capture functional and non-functional requirements for the pharmacy migration.

3. **Phase 3 – Cloud Architecture (Weeks 3–4)**  
   Design and document the AWS architecture, including all core services and data flows.

4. **Phase 4 – Cloud Implementation (Weeks 5–6)**  
   Build the solution in AWS, configure networking, servers, databases, and backups, and capture evidence via screenshots.

5. **Phase 5 – Costing and Reflection (Weeks 7–8)**  
   Estimate monthly running costs using the AWS Pricing Calculator and reflect on maintenance, scaling, and limitations.
