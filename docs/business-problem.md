# Business Problem

A medium-sized UK pharmacy chain operates four branches. Each branch uses its own local computer and spreadsheets to record prescriptions and track medicine stock. These systems are not connected.

This causes several issues:

- Staff cannot see stock levels in other branches and must call around manually to find medicines.
- If a local computer fails, patient and stock data may be lost permanently.
- Sensitive prescription data is stored in basic files, which is a privacy and security risk.

The business needs a central, cloud-based system that:

- Stores all patient and inventory data in one place
- Shows real-time stock levels across all branches
- Reduces the risk of data loss and improves data security
- Supports compliance with UK data protection laws such as GDPR
